Eat Away App Showcase

This is a restaurant selector app designed to help users decide where to eat. It takes into account location, budget, and lets the user select foods that they like.

View the source code on GitHub.

Homepage

This is the homepage of the application. It purpose is to direct you to the two other tabs where the functionality of the app is.

Select City

Here the user enters the city name and state and hits search and it returns a list of possible matches to select from.

Choice Price Level

Here the user selects their desired price level for eating out.

Select Meal

Here the user selects which type of meal they are looking for.

Select Food by Type of Food

The user has two ways to decide on food here they can select by food type.

Select Food by Region

The second way to decide on a type of food is selct by differt regions

Favorites

The user can also save restaurants as favorits which end up here.
